# 公文轨范 · Gongwen Guifan

> 在浏览器本地运行的中国公文智能排版工具，支持 Markdown 清理、标题层级识别与 Word 导出。

公文轨范面向需要快速整理中文公文版式的个人和团队。将原始文稿粘贴进页面，即可按常用 A4 公文参数生成可预览、可打印、可导出的排版结果。

## 特性

- 单文件网页，无需安装依赖或搭建服务，直接打开 `index.html` 即可使用。
- 支持选择字体和中文字号名称，并可设置固定行距（磅）、页边距及段后间距。
- 默认提供标题二号方正小标宋简体、正文三号仿宋、一级黑体、二级楷体等常用配置；所有字体均可自行调整。
- 自动识别标题、抬头、落款及一至四级标题：`一、`、`（一）`、`1.`、`（1）`。
- 自动处理“一是、二是、三是”等条目：短句整行作为加粗三级标题；较长段落仅加粗引导语。
- 自动识别同一行连续的 `**一是……。二是……。**`，拆分为独立标题并首行缩进两字符。
- 可清理 AI 常见 Markdown 标记，如 `#`、`*`、`**`、引用、列表和链接。
- 识别“民政局：”“各部门：”等抬头，不作首行缩进。
- 自动在标题后和落款前保留一个正常空白段落。
- 支持日间／夜间模式、A4 预览、打印／导出 PDF、TXT、DOC 和 DOCX。

## 快速开始

1. 下载或克隆本仓库。
2. 用任意现代浏览器直接打开 [index.html](./index.html)。
3. 在左侧粘贴文稿；网页会自动清理可识别的 Markdown 并生成右侧 A4 预览。
4. 需要交付时，选择 `DOCX` 或 `DOC`，点击“导出 Word”。

网页完全是静态文件，也可部署到 GitHub Pages、Netlify、Vercel 或任意静态网站托管服务。

## 默认排版参数

| 项目 | 默认值 |
| --- | --- |
| 纸张 | A4（210 × 297 mm） |
| 正文 | 三号仿宋 |
| 标题 | 二号方正小标宋简体 |
| 一级标题 | 黑体 |
| 二级标题 | 楷体 |
| 三级、四级标题 | 仿宋加粗 |
| 固定行距 | 28 磅 |
| 页边距 | 上 37 mm、下 35 mm、左 28 mm、右 26 mm |

以上为常用公文版式起点。正式发文前，请按本单位的具体制度、模板和审签要求复核内容与格式。

## 隐私与数据

公文轨范不需要账号，也不会将文稿发送至服务器。文本识别、排版和 DOC/DOCX 生成均在当前浏览器内完成。

为便于继续编辑，网页会将当前文稿和排版设置写入**当前设备的浏览器本地存储**。这些数据不会上传或与第三方共享；如需清除，可在浏览器的网站数据设置中删除本页面的本地数据。

## 导出说明

- **DOCX**：在浏览器本地生成 Office Open XML 文件，建议优先使用。
- **DOC**：导出为 Word 可打开的 HTML 文档，适合兼容性需求。
- **PDF**：使用“打印 / 导出 PDF”，由浏览器的打印对话框生成。

不同系统和 Word/WPS 版本安装的中文字体并不完全一致。若目标设备缺少所选字体，软件会使用系统可用字体回退；重要文件建议在交付环境中复核一次版式。
